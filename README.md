# The Plant Theory 🌿

An e-commerce web application for buying plants online.

## Features
- User authentication (Login/Register)
- Product catalog
- Shopping cart (session-based)
- Admin panel
- Order management

## Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript

## Setup Instructions
1. Install XAMPP
2. Import plant_theory.sql into phpMyAdmin
3. Place project in htdocs
4. Run on localhost
